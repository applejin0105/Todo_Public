## To Do App

<br>

### 참고사항
- 사용한 아이콘은 Anime-Girls-Holding-Programming-Books 프로젝트의 다음 경로에 있는 사진을 배경만 제거하여 사용하였습니다.
-     https://github.com/cat-milk/Anime-Girls-Holding-Programming-Books/blob/master/C%2B%2B/cirno_teaches_c%2B%2B.jpg
- 글씨체는 D2Coding을 사용하였습니다.
-     https://github.com/naver/d2codingfont
- 기능은 다음과 같습니다.

<br>

<img width="900" height="700" alt="Image" src="https://github.com/user-attachments/assets/1f823147-24a7-464a-b8dd-1d2c0ba605e8" />

- 가장 왼쪽부터 저장할 이름 / 플랫폼 / 시작 날짜 / 시작 시간 / 마감 날짜 / 마감 시간 입니다. 입력 후 Add 버튼을 누르면 추가됩니다.
- 마감일/시작일 기준으로 정렬이 가능하며 Platform Management 창을 통해 플랫폼 목록을 추가할 수 있습니다.
- 날짜는 자체적인 달력 아이콘을 눌러 설정하거나, 2025/08/05 혹은 2025-08-05 형식으로 입력 가능합니다.
- 각 항목은 Not Started / In Progress / Completed로 설정이 가능하며, Edit과 Delete가 가능합니다.
- Setting 버튼을 통해 설정이 가능합니다.

<br>

<img width="400" height="450" alt="Image" src="https://github.com/user-attachments/assets/386e42c1-9ade-4ddf-93e5-30e28a84b553" />

- 플랫폼 관리 창에서는 원하는 플랫폼을 입력 후 Add 하면, Main 창의 플랫폼 선택 드롭박스에 적용됩니다.
- Delete를 통해 삭제 가능합니다.

<br>

<img width="400" height="600" alt="Image" src="https://github.com/user-attachments/assets/f5594d59-f9d9-4841-b025-16c3c8ee6781" />

- Setting 창에서는 테마 선택 / 항상 최상단 표시 / PC 시작시 자동 재시작 / 윈도우 알림 받기 기능을 설정할 수 있습니다.
- 카카오톡 로그인이 가능합니다.
- 윈도우 알림 테스트를 지원합니다.
- 알림은 리눅스 서버와 연동하여 새로운 Task 추가 시 카카오톡 알림, 매일 아침 및 저녁 브리핑 등 알림이 나옵니다.
- 윈도우 알림은 그날 해야할 일(In Progress 항목), 작업 종료 1시간전 작업 알림이 나옵니다.
